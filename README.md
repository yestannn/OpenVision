# OpenVision

### Installation

1. Dowload Python3 (https://www.python.org)
2. Clone the repo
   ```sh
   git clone https://github.com/yestannn/.git
   ```
3. Install pip   
   ```
   python get-pip.py  (for mac python3 get-pip.py)
   ```
   ubuntu:
   ```
   sudo apt update
   sudo apt install python3-pip
   ```
 
4. Install Django
   ``` 
   pip install Django (for mac and ubuntu pip3)
   ```
